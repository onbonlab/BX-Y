# 4.3.13   定时调亮 - customBrightness

*指令样例：*

```json
{
  "protocol": {
     "name": "YQ-COM2","version": "1.0",
     "remotefunction": {
      "name": "customBrightness"
      "input": {
        "items":["1","2","3",...,"48"]
      }
    }
  }
}
```

参数说明：

* "items"： 48个范围1～255的亮度值，对应全天48个半小时的亮度*指令样例：*
